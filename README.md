# TaskScheduler - Планировщик задач

**TaskScheduler** - это простое и удобное приложение для управления повседневными задачами. Оно помогает организовать рабочий процесс, устанавливать сроки выполнения, приоритеты и отслеживать прогресс.

## Возможности

-  Создание, редактирование и удаление задач
-  Установка дедлайнов и напоминаний
-  Приоритеты (низкий, средний, высокий)
-  Категории / проекты для группировки задач
-  Отметка о выполнении
-  Поиск и фильтрация по статусу, сроку, приоритету
-  Статистика и прогресс выполнения

## Технологии

Проект построен на следующем стеке:

- **Backend**: Python (FastAPI) 
- **Frontend**: HTML+CSS+JS 
- **База данных**: PostgreSQL
